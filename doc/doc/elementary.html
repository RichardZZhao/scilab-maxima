INDEX FUNCTIONS

All the functions presented here are considered element-wise if applied to a matrix.

Of course, the results below depend on the real declaration. 

   syms a b real
   c = a + %i*b;
   conj(c)

   ans  =

   a-%i*b  // Result depending on Maxima version

   real(c)

   ans  =

   a

   imag(c)

   ans  =

   b

   abs(c)

   ans  =

   sqrt(b^2+a^2)

Remark: the function exp still has several bugs. 

   syms a b real
   c = a + %i*b;
   log(c)

   ans  =

   log(%i*b+a)

   exp(ans)

   ans  =

   %i*b+a

Lets us consider rotation matrices. Multiplying rotation matrices corresponds to adding angles as checked now. 

   syms a b real
   Ma = [cos(a) sin(a);-sin(a) cos(a)];
   Mb = [cos(b) sin(b);-sin(b) cos(b)];
   Ma*Mb

   ans  =

   !cos(a)*cos(b)-sin(a)*sin(b)   cos(a)*sin(b)+sin(a)*cos(b)  !
   !                                                           !
   !-cos(a)*sin(b)-sin(a)*cos(b)  cos(a)*cos(b)-sin(a)*sin(b)  !

   simple(ans)

   ans  =

   !cos(b+a)   sin(b+a)  !
   !                     !
   !-sin(b+a)  cos(b+a)  !

   simple(Ma'*Ma)

   ans  =

   !1  0  !
   !      !
   !0  1  !
