INDEX FUNCTIONS

The function latex returns a LaTeX form of a symbolic expression. The function fortran returns the Fortran code corresponding to a symbolic expression. 

   syms a b c
   M = [a^2+3 0 b*c;1 c^3 b;1 2/a 3/(1+4*b)];
   latex(M)

   ans  =

   $$\begin{pmatrix}a^2+3&0&b\,c\cr 1&c^3&b\cr 1&{{2}\over{
      a}}&{{3}\over{4\,b+ 1}}\cr \end{pmatrix}$$

   fortran(M)

   ans  =

   !      M(1,1) = a**2+3     !
   !                          !
   !      M(1,2) = 0          !
   !                          !
   !      M(1,3) = b*c        !
   !                          !
   !      M(2,1) = 1          !
   !                          !
   !      M(2,2) = c**3       !
   !                          !
   !      M(2,3) = b          !
   !                          !
   !      M(3,1) = 1          !
   !                          !
   !      M(3,2) = 2/a        !
   !                          !
   !      M(3,3) = 3/(4*b+1)  !

Other functions:

  • hlp mimics the Matlab help function: displays a function help in the command window. If you know that the considered function is overloaded, type hlp sym/name_of_function directly, otherwise, hlp name_of_function returns the list of the corresponding overloaded functions.
  • typ mimics the Matlab type function: displays a function file in the command window. Same remark as above for overloaded functions.
  • hlp symbolic lists the functions of the toolbox.
