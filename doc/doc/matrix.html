INDEX FUNCTIONS

We have already introduced symbolic matrix representations. Here we introduce a few functions. 

// Definition of a symbolic matrix

   syms a b c
   M = [a^2+3 0 b*c;1 c^3 b;1 2/a 3/(1+4*b)]

   M  =

   !a^2+3  0    b*c        !
   !                       !
   !1      c^3  b          !
   !                       !
   !1      2/a  3/(4*b+1)  !

// Determinant

   det(M)

   ans  =

   (a^2+3)*(3*c^3/(4*b+1)-2*b/a)+b*c*(2/a-c^3)

// Inversion followed by a simplification

   N = inv(M);
   simple(M*N)

   ans  =

   !1  0  0  !
   !         !
   !0  1  0  !
   !         !
   !0  0  1  !

// Eigenvalues (warning: see hlp sym/eig)

   syms a b
   P=[a a^2;b 3];
   val = eig(P)

   val  =

   !(-sqrt(4*a^2*b+a^2-6*a+9)+a+3)/2  !
   !                                  !
   !(sqrt(4*a^2*b+a^2-6*a+9)+a+3)/2   !

Other functions
  • diag, retrieves the diagonal of a square symbolic matrix or, if applied to a symbolic vector, generates a square diagonal matrix.
  • trace computes the trace of a symbolic matrix.
  • size, can be invoked as n1n2 = size(M), n1 = size(M,1), n2 = size(M,2), [n1,n2] = size(M).
  • null, computes the null space.
  • rank, computes the rank.
  • colspace, computes the column span.
  • rref, computes the row echelon form of a matrix (Gaussian elimination).
  • triu and tril compute upper and lower triangular forms of a symbolic matrix.
