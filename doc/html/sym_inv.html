FUNCTIONS

Function: %sym_inv

Purpose

Inverts a symbolic matrix using the Maxima function invert.

Synopsis

  • [B] = inv(A)

Input arguments

A Symbolic square matrix.

Output arguments

B Symbolic square matrix.

See also: sym/det, sym/rank, sym/rref.

EXAMPLE
    A = sym(['a-b','a-b';'2','a+b'])
    B = inv(A)
    simple(B)
