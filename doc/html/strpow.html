FUNCTIONS

Function: strpow

Purpose

Scalar power of a scalar.

This is used to replace scalar symbolic powers for intermediate calculus (symbolic powers call Maxima, so, needs a lot of CPU time).

Synopsis

  • [cc] = strpow(aa,bb)

See also: strminus, stradd, strvconcat, strhconcat.

EXAMPLE
   strpow('a',2)
