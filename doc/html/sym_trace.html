FUNCTIONS

Function: %sym_trace

Purpose

Computes the trace of a symbolic square matrix

Synopsis

  • [B] = trace(A)

Input arguments

A Symbolic square matrix

Output arguments

B Symbolic expression

See also: sym/eig, sym/diag, sym/triu, sym/tril.

EXAMPLE
    syms a b c d
    M = [a^5*b 4;a*b*c^3*d a*d]
    trace(M)
