FUNCTIONS

Function: mexec

Purpose

Executes a string in Maxima and retrieves Maxima response as a string. For retrieving a symbolic object from Maxima workspace use preferably the function 'maxima'.

Synopsis

  • [resp] = mexec(str1[,flag])

Input arguments

str1 String in Maxima syntax like 'conjugate(a+%i*b)'
flag String equal to 'str' or 'vec'(default: flag = 'vec'). This flag sets the type of the output argument as:

- case 'str' -> concatenated string

- case 'vec' -> vector of strings

Output arguments

resp String.

See also: maxima, str2max2sym.

EXAMPLE
   resp = mexec('%pi-b$ cos(%)')
   mexec('load(ntrig)')
