FUNCTIONS

Function: %sym_det

Purpose

Computes the determinant of a symbolic square matrix

Synopsis

  • [B] = det(A)

Input arguments

A Symbolic square matrix

Output arguments

B Symbolic expression

See also: sym/eig, sym/diag, sym/trace, sym/inv.

EXAMPLE
    syms a b c d
    M = [a^5*b 4;a*b*c^3*d a*d]
    det(M)
