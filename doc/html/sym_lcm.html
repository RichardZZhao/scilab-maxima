FUNCTIONS

Function: %sym_lcm

Purpose

Computes the least common multiple of two polynomials.

Synopsis

  • [ll] = lcm(aa,bb)

Input arguments

aa Symbolic polynomial
bb Symbolic polynomial

Output arguments

ll Least common multiple symbolic polynomial

See also: sym/gcd.

EXAMPLE
    syms x y
    p1 = y^3+x;
    p2 = x^5+x^3+4*x^2+4;
    l = lcm(p1,p2)
