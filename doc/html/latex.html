FUNCTIONS

Function: latex

Purpose

Transforms a symbolic expression to a string compatible with LaTeX.

Synopsis

  • [str] = latex(expr)

Input arguments

expr Symbolic expression.

Output arguments

str LaTeX string corresponding to 'expr' compatible with amsmath style.

See also: fortran.

EXAMPLE
    syms a b c d
    M = [a^5*b 4;a*b*c^3*d a*d]
    latex(M)
