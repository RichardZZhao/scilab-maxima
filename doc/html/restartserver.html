FUNCTIONS

Function: restartserver

Purpose

Restarts Maxima and the server used for communication between Scilab and Maxima (Symbolic toolbox). In case Scilab freezes you must kill manually maxima and perl/serveur tasks.

Synopsis

  • restartserver(boolean)

Remark

This function should be used in the following cases:

- You forgot to invoke killserver before the last time you shut down Scilab.

- You have just invoked the scripts builder.sce and loader.sce of the symbolic toolbox.

- You want to re-initialize Maxima workspace.

Caution

Windows: For killing tasks, we use ProKill v2.1 (pk.exe). All tasks named perl.exe or maxima.exe will be killed, so be careful.
See: http://www.white-scorpion.nl/programs/programs.html.

Input arguments

boolean If boolean = %t no comments are displayed. Default boolean = %f.

See also: killserver, findpid.
