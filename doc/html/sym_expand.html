FUNCTIONS

Function: %sym_expand

Purpose

Expands a rational expressions using the Maxima function expand

Synopsis

  • [B] = expand(A)

Input arguments

A Rational (possibly matrix) symbolic expression.

Output arguments

B Expanded symbolic expression

See also: sym/simple, sym/factor.

EXAMPLE
    A = sym('3*(a+b+c)^3-(a+b-c)^3')
    B = expand(A)
