FUNCTIONS

Function: declare

Purpose

Declare types of atoms in Maxima workspace. This function is invoked by other functions of the toolbox.

Synopsis

  • declare(aa,bb,string,nn,string,cc,string,...)

Input arguments

aa, bb, cc, nn Symbolic ATOMS
string Strings = even, odd, integer, rational, irrational, real, imaginary or complex

See also: syms, symtype.

EXAMPLES
    syms a b c d
    declare a b 'real' c d 'imaginary'
    symtype(a)
    symtype(d)
