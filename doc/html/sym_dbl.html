FUNCTIONS

Function: %sym_dbl

Purpose

Converts symbolic numbers to floating point numbers. Note that in some cases this function looks similar to eval but the output of eval is a symbolic object (possibly symbolic number).

Synopsis

  • [bb] = dbl(aa)

Input arguments

aa Symbolic expression containing no symbolic atoms.

Output arguments

bb Scalar or matrix of floating point numbers

See also: bfloat, float, sym/eval.

EXAMPLE:
    a = log(sym('3'))
    float(a)        // Symbolic floating point number
    dbl(a)          // Numerical floating point number (indented)
