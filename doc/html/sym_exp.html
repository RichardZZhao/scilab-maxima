FUNCTIONS

Function: %sym_exp

Purpose

Element-wise exponential of a symbolic matrix

Synopsis

  • [A] = exp(B)

Caution

The part of the function max2scistr concerning the function exp must be rewritten for supporting expressions like %e^-(...) and so on. 

EXAMPLE:
 syms a b c d
 M = [a^5*b 4;a*b*c^3*d a*d]
 P = exp(M)
 P^2
