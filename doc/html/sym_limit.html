FUNCTIONS

Function: %sym_limit

Purpose

Computes the limit of a symbolic expression using the Maxima function limit

Synopsis

  • [B] = limit(A,xx,xx0[,sign])

Input arguments

A Rational (possibly matrix) symbolic expression.
xx Symbolic atom appearing in A. The limit of A is considered w.r.t. xx.
xx0 Constant. Value of xx at which the limit is evaluated.
sign String equal to '-' or '+', indicates left or right limit.

Output arguments

B Limit in symbolic form.

See also: sym/diff, sym/integ.

EXAMPLE
    syms a
    limit(cos(a),a,%pi/4)
