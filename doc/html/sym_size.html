FUNCTIONS

Function: %sym_size

Purpose

Computes the size of a sym-object

Synopsis

  • [n1n2] = size(A)
  • [n1,n2] = size(A)
  • [ni] = size(A,ii)

Input arguments

A Symbolic matrix
ii integer equal to 1 or 2 

EXAMPLE
   M = sym(['1' 'a' '2';'3' 'b' 'c'])
   n1n2 = size(M)
   [n1,n2] = size(M)
   size(M,1)
   size(M,2)
