Purpose

Transforms a string in Maxima syntax to Scilab syntax


Synopsis
  • [str2] = max2scistr(str1)


Description

Treats in particular matrices. If a function is applied to a matrix, say cos(matrix([...])) this function is applied element by element (provisional). For improving this feature, the string associated to a symbolic object (a.str1) must be replaced by a list or a series of strings.

Input arguments
  • str1 :   String compatible with Maxima

Output arguments
  • str2 :   String compatible with Scilab (used for display or evaluation)


See also: scistr2max, str2max2sym.
