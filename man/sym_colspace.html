Purpose

Null space of a symbolic matrix using the Maxima function colspacespace.


Synopsis
  • [I,warn] = colspace(A)

Input arguments
  • A :   Symbolic matrix

Output arguments
  • I :   Symbolic matrix: column space.
  • warn :   String giving restrictions on column span validity.


See also: sym/rank, sym/null.
EXAMPLES
   syms a b
   M = [a 2*a 3*a;b 2*b 3*b];
   [I,warn] = colspace(M)
