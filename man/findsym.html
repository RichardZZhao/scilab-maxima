Purpose

Extracts the symbolic atoms from a symbolic expression using the Maxima function listofvars.


Synopsis
  • [vars] = findsym(expr)

Input arguments
  • expr :   Symbolic expression

Output arguments
  • vars :   Symbolic vector.


See also: numden.
EXAMPLE
    syms a b c d
    M = [a^5*b 4;a*b*c^3*d a*d]
    findsym(M)
