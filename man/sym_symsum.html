Purpose

Series summation


Synopsis
  • [sym1] = symsum(A,nn,n_min,n_max)

Input arguments
  • A :   Symbolic expression
  • nn :   Integer symbolic variable for series summation
  • n_min :   First value of nn for summation
  • n_max :   Last value of nn for summation


See also: sym/int.
EXAMPLE
    syms a n
    symsum(a^(-n),n,0,10)
    symsum(a^(-n),n,0,%inf)
