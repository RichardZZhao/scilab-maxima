Purpose

Kill all maxima tasks and the server lauched for communication between Maxima and Scilab. this function must be invoked manually before shuting down scilab. For re-initialization of Maxima invoke the function restartserver.


Synopsis
  • killserver(boolean)


Caution

Windows: For killing tasks, we use ProKill v2.1 (pk.exe). All tasks named perl.exe or maxima.exe will be killed, so be careful.
See: http://www.white-scorpion.nl/programs/programs.html.

If you forgot to invoke killserver before shuting down Scilab, next time you launch Scilab, invoke the function restartserver before you invoke any function of the symbolic toolbox.

Input arguments
  • boolean :   If boolean = %t no comments are displayed. Default boolean = %f.


See also: restartserver, findpid.
