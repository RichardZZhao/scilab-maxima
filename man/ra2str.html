Purpose

Utility for evaluation from values in the workspace of rational expressions .


Synopsis
  • [str,var] = ra2str(r_expr)

Input arguments
  • r_expr :   Scilab standard rational expression

Output arguments
  • str :   String which can be evaluated 

EXAMPLE

   r_expr = [%s 3*%s 1/%s;2+1/(1+3*%s) 2 %s];
   [str,var] = ra2str(r_expr)
