Purpose

Retrieves numerators and denominators of a rational symbolic matrix using the Maxima functions matrixmap, ratnumer and ratdenom.


Synopsis
  • [num,den] = numden(A)

Input arguments
  • A :   Rational symbolic expression

Output arguments
  • num :   Symbolic expression
  • den :   Symbolic expression


See also: coeffs, degrees, findsym.
EXAMPLE
   syms a b
   M = [2*a/(1+b) 0;1 a/b];
   [num,den] = numden(M)
