Purpose

Creates symbolic atoms in Scilab workspace and declares types (as real, odd,...) of these atoms in Maxima workspace.


Synopsis
  • syms aa bb string nn string cc string
  • syms aa bb cc


Description

By default, non already typed atoms are considered as complex (Maxima 5.9.2 or higher) or real (Maxima 5.9.1). If a type was already declared and if a new type is not specified when syms is invoked, the previously defined type is preserved.

Input arguments
  • aa, bb, cc, nn :   Strings (names of symbolic atoms)
  • string :   Strings: even, odd, integer, rational, irrational, real, imaginary, complex or default.


See also: sym, symtype.
EXAMPLES
    syms a b real c d imaginary
    symtype(a)
    symtype(d)
