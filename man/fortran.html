Purpose

Transforms a symbolic expression to a string compatible with LaTeX.


Synopsis
  • [str] = fortran(expr[,M])

Input arguments
  • expr :   Symbolic expression.
  • M :   String correponding to the name of the expression expr. If expr is a matrix, this second argument is required, its value is 'M' by default.

Output arguments
  • str :   Fortran string corresponding to 'expr'


See also: latex.
EXAMPLES
    syms a b c d
    fortran(a^2+0.3*b/(0.5+c^2)^2)

    X = [a^5*b 4;a*b*c^3*d a*d]
    fortran(X,'M')
