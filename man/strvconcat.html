Purpose

Vertical concatenation of strings similar to symbolic vertical concatenation. This function is used to replace symbolic vertical concatenation for intermediate calculus (symbolic vertical concatenation calls Maxima, so, needs a lot of CPU time).


Synopsis
  • [cc] = strvconcat(aa,bb)


See also: stradd, strminus, strmult, strhconcat.
EXAMPLE
   strvconcat(['a','b';'1','2'],['1','2';'a','b'])
