Purpose

Null space of a symbolic matrix using the Maxima function nullspace.


Synopsis
  • [X,warn] = null(A)

Input arguments
  • A :   Symbolic matrix

Output arguments
  • X :   Symbolic matrix
  • warn :   String giving restrictions on null space validity.


See also: sym/rank, sym/colspace.
EXAMPLES
   syms a b
   M = [a 2*a 3*a;b 2*b 3*b];
   [X,warn] = null(M)
