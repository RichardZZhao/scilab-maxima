Purpose

Evaluation of a symbolic expression from values of the symbolic atoms in Scilab workspace. The atom values in the workspace can be constant, polynomial, symbolic.


Synopsis
  • [B] = eval(A)


Remark

Needs to be revisited for a special treatment of %pi, %e and %i.

Input arguments
  • A :   Symbolic expression

Output arguments
  • B :   Evaluated form of A.


See also: subs, float, bfloat, sym/dbl.
EXAMPLE
    syms a b c d e f
    A = sym(((a+b+c)^2/(d+e+f)))
    d = a; e = b; f = c;
    B = eval(A)
    a = 2; b = 3;
    C = eval(B)
