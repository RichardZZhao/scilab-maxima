Purpose

Factorization of rational expressions using the Maxima function factor.


Synopsis
  • [B] = factor(A)

Input arguments
  • A :   Rational symbolic expression

Output arguments
  • B :   Factorized symbolic expression


See also: sym/expand, sym/simple.
EXAMPLE
    A = sym('((a+b+c)^2-(d+e+f)^2)/(a^2-b^2)');
    A = expand(A)
    B = factor(A)
