Purpose

Computes the row echelon form of a matrix (Gaussian elimination) using the Maxima function echelon.


Synopsis
  • [B] = rref(A)

Input arguments
  • A :   Symbolic matrix

Output arguments
  • B :   Symbolic matrix


See also: sym/triu, sym/tril.
EXAMPLES
   syms a b
   M = [1 2*b 3*a;2 3*b 0;a*b 0 1];
   B = rref(M)
