Purpose

Rank of a symbolic matrix


Synopsis
  • [nn,warn] = rank(A)

Input arguments
  • A :   Symbolic matrix

Output arguments
  • nn :   Integer
  • warn :   String giving restrictions on rank result


See also: sym/null, sym/colspace.
EXAMPLES
   syms a b
   M = [a 2*a 3*a;b 2*b 3*b];
   [n,warn] = rank(M)
