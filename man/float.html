Purpose

Converts integers, rational numbers and bigfloats in input argument to floating point numbers.


Synopsis
  • [bb] = float(aa)

Input arguments
  • aa :   Symbolic expression or expression which can be converted into a symbolic expression (e.g., a string) before its numerical evaluation is returned in b.

Output arguments
  • bb :   Symbolic expression or symbolic number


See also: bfloat, sym/dbl.
EXAMPLES:
    syms b
    %pi*b
    float(%pi*b)

    float('(1/3)*a')
